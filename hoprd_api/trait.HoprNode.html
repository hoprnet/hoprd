Expand description

Combined trait bound for the HOPR node type parameter used throughout the REST API.

Any type H: HoprNode can be used as the node implementation backing the API. In production this is Hopr<Chain, Graph, Net, TMgr>; in tests it can be a mock.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
